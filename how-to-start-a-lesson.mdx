<Steps>
  <Step title="From the Skills page">
    1. Select the skill you would like to work on
    2. Select a lesson to complete
    3. Select "Start Lesson"
  </Step>
  <Step title="Completed!">
    You have now successfully completed a lesson!
  </Step>
</Steps>

<Info>
  Please note that the color of the lesson shows you the following information:

  🟩 = Completed

  🟦 = Available

  ◻️ = Not Available Yet/Locked
</Info>
