#### Fixed Bugs

- When clicking on a trip already created from explore view, does not go to completed view.
- When there is a pending friend request, user is still searchable
- There is no notice when no results are found from friend search
- Friend search is case sensitive
- Top 10 adventure automatically closes when ten categories filled
- "Done" button on premium upgrade page is green when it should be purple
- When searching for friends there is no indicator that a search is happening

#### New Features

- Photos are now checked to make sure they are appropriate before uploading to Top 10.
- Users now have the ability to enter a promo code while signing up for Top 10 Pro.
- Users now have the ability to turn on/off saving photos locally to their iPhones
- You can now go to topten://event-center in your browser to go directly to the latest event
- General feed loads faster (efficiency improvements)
- You can now scroll between images in the Photo Capture View
- Users can now tap to dismiss the keyboard on both the capture page and the friends view
- Users can now see the current version number installed in settings
