#### **Fixed Bugs**

- Support link in settings now works
- Occasionally when no results would be found from filters, default error message would show
- The phone number was not formated under filters
- Scrollable feed cards only worked in light mode
- Suggested friends would never finish loading occasionally
- The No Picture message on a friend profile was not centered
- Occassionally pre-loading in the app would stop when switching pages
- Upgrade to pro occassionally would not work

#### **New Features**

- You can now search by phone number, username, or name to add friends
- Filters now work even faster with a new refreshed UI
- Updated loading UI to match the new modern UI
- Updated the completed trip view to match refreshed UI
- Feed cards have been updated with a new look to be more modern and support both light and dark modes
- You can now share photos from the feed view externally
- New and Improved In App Settings Menu
- You can now set the theme of the app in settings from light to dark or system
- New Upgrade to Pro page
- Added full name to profile to make it easier to find friends
- Updated delete account page
- Added the ability to comment on photos

#### **Improvements**

- Internal speed enhancements
- Internal system enhancements
