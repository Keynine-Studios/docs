#### **Fixed Bugs**

- Duplicate places show in the filters twice occasionally
- Tab notifications are sometimes not accurate
- Reactions for events do not save
- App allows you to add locations more than once in some instances
- All information boxes on profile page are not the same size.
- Profile picture lags when saving
- Feed does not load fast
- All pictures on friend's profile page aren't coming through
- Menu on the friend's profile page comes from center, not top right

#### **New Features**

- Total UI Redesign using Apple's Native Liquid Glass
- You can now create trip links to share for non-complete trips.
- You can now invite friends even if they aren't on the app yet!
- You can now rate the app via the settings menu on the profile page.
- You may now get occaasional notifications from the app
- The "Recent Cities" section of the profile page has been removed
- You can now see what trip a photo was from in the feed view
- The duration has been replaced with a date/time stamp on the feed view
- You can now download the app directly from the link people share with you
- The explore page has been completely re-done and modernized with the new "Popular Destinations" section
- The main menu bar has been switched from a custom system to liquid glass
- You can now get AI Suggestions for ANY category in ANY location!
- You can now recategorize photos in the image capture view.
- The profile view has been completely revamped!
- Updated password reset to verify user's identity with a SMS verification code.
- Search arrow added to friends page
- Friend suggestions are now it's own page
- Event's center now also uses liquid glass
- Added a new deep link to specific images
- Added "Dark Mode" to new single image view

#### **Improvements**

- Friend Profile's load even faster!
- Moved the delete button in the capture view to avoid accidental deletions
- Suggested friends now loads faster!
- The badges section on a profile page only appears if badges are present.
- Changed from the temp password when user forgot to change password dialogue on the login screen.
- Removed the word "caption" from the category photo page to decrease redundancy.
- Updated all feeds to use the same logic as the main feed.
- Reaction icons modernize
- Updated the color gradients for the Terms and Privacy page
- Login and signup pages redesigned to match the app better
- Moved settings into the settings icon on the top right of the profile picture from the bottom of the profile page
- Padding added back into profile page
- Added cancel button when in profile edit mode
- Updated version number in settings
- Added dark mode for skeleton loading
- Updated profile page cards to match other cards in app
